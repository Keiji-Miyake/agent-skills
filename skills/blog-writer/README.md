# Blog Writer Skill

収集された素材を元に、高品質な技術ブログを書き上げる「執筆・編集」専門エージェント。

**Version**: 1.1  
**Author**: Keiji Miyake

## 概要

`blog-writer` は、メモや技術的な事実を、読者の心に届く「記事」へと昇華させます。SEOの最適化から、読みやすい構成の提案、そして視覚的な図解（Mermaid）の作成まで、プロフェッショナルな編集者の役割を担います。

## インストール方法

```bash
npx skills add Keiji-Miyake/agent-skills --skill blog-writer
```

## 主な機能

- **高品質な執筆**: ターゲット読者に合わせた最適なトーンでの文章作成。
- **構成案の提案**: 理路整然とした章立ての設計。
- **SEO 最適化**: 検索エンジンと読者の双方にアピールするメタ情報の生成。
- **Mermaid 図解**: 文章を補完する図解案の提示。

## 組み合わせ例

手動で作成したメモ、または履歴解析ツールによって収集された素材を組み合わせて、効率的に執筆を進めることができます。
