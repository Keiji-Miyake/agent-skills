# Dev Ready

開発開始前の準備チェックを行うスキルです。

## 概要

開発セッション開始時にプロジェクトの現状を把握し、次のアクションを提案します。

## 機能

- Gitステータス・コミット履歴の確認
- タスク管理ファイル (TASKS.md, TODO.md等) の確認
- docs/ ディレクトリ内のドキュメント確認
- 会話履歴・ナレッジアイテムの確認
- Feature/Skills別の進捗確認
- 現状サマリーと次のアクション提案

## 使い方

このスキルがインストールされている状態で、以下のように依頼してください：

```
開発を始める前にプロジェクトの状態を確認してください
```

または

```
dev-readyスキルを使って現状を確認してください
```

## 比較

このスキルは `.agent/workflows/dev-ready.md` と同等の機能を提供します。

| 項目 | Workflow版 | Skill版 |
|------|-----------|---------|
| 配布方法 | リポジトリ内同梱 | `npx skills add` |
| 呼び出し | `/dev-ready` | 自然言語で依頼 |
| 互換性 | Gemini CLI固有 | Skills対応ツール全般 |
