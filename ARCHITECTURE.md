# Architecture (アーキテクチャ)

本リポジトリは、AIコーディングアシスタント向けの再利用可能なエージェントスキルのコレクションです。

## リポジトリ構造

```
agent-skills/
├── .agent/              # エージェントの設定と指示
│   └── instructions.md  # プロジェクト固有のルール
├── docs/                # プロジェクトドキュメント (dev-support準拠)
│   ├── README.md        # ドキュメントインデックス
│   └── dev/             # 機能開発コンテキスト
├── scripts/             # 開発・検証用スクリプト
├── skills/              # 提供されるスキル（本体）
│   └── [skill-name]/
│       ├── SKILL.md     # AIエージェント向け指示
│       └── README.md    # 人間向け説明
└── package.json          # 依存関係とバリデーションスクリプト
```

## スキルの設計原則

1. **ポータブル**: 特定のAIエージェントに依存せず、Agent Skills仕様に準拠する。
2. **トークン効率**: `SKILL.md` は簡潔に保ち、詳細は `references/` や外部ドキュメントに委ねる。
3. **検証可能性**: 全てのスキルは `npm run validate` によって形式と構造がチェックされる。

## 日本語化の方针

- 日本語での利用を促進するため、メインドキュメントには日本語版（`.ja.md`）を用意する。
- 開発コンテキストやプロジェクト内ドキュメントは日本語を基本とする（`.agent/instructions.md` による制御）。

## ドキュメントの成熟度レベル

現在のプロジェクトレベル：**Level 2 - Growing**
- `ARCHITECTURE.md` によって全体構造が定義されている。
- `docs/dev/` によって開発コンテキストが管理されている。
